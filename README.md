# Osu! Map Mixer

A graphical interface tool for editing and combining beatmaps for osu! as if they are "videos".

## Features

- Video editor-like interface for beatmap editing
- Canvas for beatmap object visualization
- Master track for timeline editing
- Import and export functionality

## Structure

- `index.html` - Main HTML structure
- `styles.css` - Styling for the interface
- `script.js` - JavaScript functionality

## Development

This is the initial layout implementation with:
1. Gray background
2. Black square canvas in the middle top
3. Master track at the bottom
4. Menu bar with Import and Export options

The interface is responsive and will be extended with beatmap rendering capabilities.
